# AC Manager

Adventarのアドベントカレンダーを管理するChrome拡張機能です。
VIBEコーティング産です。

## 機能

- 📅 **カレンダー登録** - Adventarのカレンダーを追加して一括管理
- 🔔 **新着通知** - 新しい記事が投稿されるとバッジで通知
- ⭐ **お気に入り** - 気になる記事をお気に入り登録
- 🔄 **自動更新** - 設定した間隔で自動的に記事をチェック

## インストール

1. このリポジトリをクローンまたはダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 使い方

### カレンダーを追加する

1. 拡張機能アイコンをクリックしてポップアップを開く
2. 入力欄にAdventarのカレンダーURLを貼り付け
   - 例: `https://adventar.org/calendars/12345`
3. 「Add」ボタンをクリック

### 記事一覧を見る

1. 登録したカレンダーをクリック
2. 投稿された記事の一覧が表示されます
3. 記事タイトルをクリックすると新しいタブで開きます

### お気に入り機能

- 記事一覧で ⭐ ボタンをクリックするとお気に入り登録
- 「Favorites」タブでお気に入り記事を確認

### 新着通知

- 新しい記事が投稿されると拡張機能アイコンにバッジが表示されます
- ポップアップを開くとどのカレンダーに新着があるか確認できます
- ポップアップを開くとバッジはクリアされます

### 設定

- 「Settings」タブで自動更新の間隔を変更できます（デフォルト: 60分）
- 「Force Update All」で手動で全カレンダーを更新
